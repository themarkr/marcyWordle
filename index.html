BETTER WORDLE
Settings Menu
Changing the value of the difficulty slider willincrease or reduce the number of attempts per game. The higher the difficulty, the less number to total attempts. ***Adjusting the value of the slider will start a new game***
Changing the Word Length will increase or reduce the number of tiles per row. ***Adjusting the value of the slider, will start a new game***
HOW TO PLAY

Guess the WORDLE in six tries.

Each guess must be a valid five-letter word. Hit the enter button to submit.

After each guess, the color of the tiles will change to show how close your guess was to the word.
